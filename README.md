# أحمد المعمري للاستشارات والخدمات الهندسية — تطبيق Streamlit

تطبيق علامة شخصية واستقبال طلبات استشارات يعمل **محلياً 100%** بدون أي واجهات برمجية خارجية.

## التشغيل

```bash
pip install -r requirements.txt
streamlit run app.py
```

ثم افتح المتصفح على: `http://localhost:8501`

## ⚠️ إعدادات إلزامية قبل الاستخدام الفعلي

| الإعداد | الملف | السطر |
|---|---|---|
| رقم واتساب الأعمال (صيغة دولية بدون +) | `app.py` | الثابت `WHATSAPP_NUMBER` |
| كلمة مرور لوحة التحكم | `pages/3_🔐_Dashboard.py` | الثابت `ADMIN_PASSWORD` |

## ملاحظات مهمة

- **البيانات:** تُحفظ الطلبات في `data/consultation_requests.db` (SQLite محلي). انسخ هذا الملف دورياً كنسخة احتياطية.
- **دراسات الحالة:** النصوص الحالية في صفحة Portfolio قوالب استرشادية معمّمة — حدّثها بوقائع أعمالك الفعلية (مع بقائها مجهّلة من أي بيانات سرية) قبل عرضها على عملاء.
- **النشر لاحقاً:** التطبيق محلي؛ لاستقبال طلبات من الإنترنت انشره على خادم أو Streamlit Community Cloud، وعندها انقل كلمة المرور إلى `st.secrets` بدل الكود.
